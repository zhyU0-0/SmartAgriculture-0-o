# 智能农业系统测试文档

## 1. 项目概述

智能农业系统是一个基于鸿蒙OS和Java Spring Boot的智能农业管理平台，旨在通过物联网技术实现对农业生产的智能化管理。系统主要功能包括用户管理、地块管理、设备管理、数据监控、天气查询、异常预警和AI聊天等。

### 1.1 系统架构

- **前端**：基于鸿蒙（ETS）开发的移动应用
- **后端**：基于Java Spring Boot框架的RESTful API服务
- **通信**：HTTP/HTTPS协议进行前后端通信
- **数据存储**：MySQL数据库
- **设备通信**：MQTT协议

### 1.2 核心功能模块

1. **用户管理**：用户注册、登录、个人信息管理
2. **地块管理**：地块创建、编辑、删除、查询
3. **设备管理**：设备添加、删除、状态控制、数据采集
4. **数据监控**：环境数据（温度、湿度、CO2等）实时监控
5. **天气查询**：实时天气和天气预报
6. **异常预警**：设备异常和环境异常预警
7. **AI聊天**：智能农业问答和指导
8. **技能管理**：技能创建、编辑、选择、使用
9. **网关管理**：网关绑定、解绑、设备管理
10. **日记管理**：笔记创建、编辑、删除、查询
11. **AI拓展功能**：施肥指导、收成预测

## 2. 测试范围

本次测试覆盖系统的主要功能模块，包括：

1. **功能测试**：验证系统各功能模块是否正常工作
2. **性能测试**：测试系统在不同负载下的性能表现
3. **安全测试**：验证系统的安全性和数据保护
4. **兼容性测试**：测试系统在不同设备和环境下的兼容性

## 3. 测试环境

### 3.1 硬件环境

- **前端设备**：鸿蒙OS设备（如华为手机、平板）
- **后端服务器**：CPU i5以上，内存8GB以上，存储空间50GB以上
- **网络环境**：局域网环境，带宽100Mbps以上

### 3.2 软件环境

- **前端**：鸿蒙OS 3.0及以上
- **后端**：Java 11，Spring Boot 2.7.0，MySQL 8.0
- **测试工具**：Postman（API测试），JMeter（性能测试），Appium（移动端测试）
